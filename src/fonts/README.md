# Fonts Directory

## Required Font Files

To use the Futura font family, please add the following files to this directory:

- `Futura-Medium.woff2`
- `Futura-Bold.woff2`

### Where to get Futura fonts:

1. **Purchase from official foundries** like MyFonts or Adobe Fonts
2. **Use system fonts** - If Futura is installed on your system, you can extract it using font conversion tools
3. **Alternative free fonts** that are similar to Futura:
   - Renner* (open source)
   - League Spartan
   - Montserrat (already included as fallback)

### Temporary Fallback

The app will currently fall back to system fonts (system-ui, arial) until Futura font files are added.

### How to add fonts:

1. Place `Futura-Medium.woff2` and `Futura-Bold.woff2` in this directory
2. Restart the dev server
3. The fonts will automatically load

## Montserrat

Montserrat is loaded via Google Fonts and doesn't require any local files.
